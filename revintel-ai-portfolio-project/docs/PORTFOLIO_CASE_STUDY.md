# Portfolio Case Study — RevIntel AI

## Summary
RevIntel AI is a revenue intelligence and sales forecasting platform built as a portfolio-grade analytics product. It turns sales data into executive KPIs, forecasts, anomaly alerts, and scenario planning outputs.

## Problem
Many business teams have raw sales data but lack a clean interface for understanding growth, forecasting revenue, spotting anomalies, and simulating pricing or promotion decisions.

## Solution
I built a full-stack analytics application with:
- executive KPI dashboards
- forecasting model comparison
- anomaly detection
- scenario planning
- CSV upload and export workflows

## Key technical work
- designed the frontend UX as a polished SaaS-style dashboard
- built backend analytics services in FastAPI
- implemented forecasting with Moving Average, Linear Regression, and Random Forest
- added rolling z-score anomaly detection
- created CSV/PDF report export support
- added lightweight persistence with SQLite

## Business value
- helps teams understand revenue performance quickly
- enables forward-looking revenue planning
- flags suspicious commercial behavior earlier
- supports pricing and campaign decision-making

## Tools
React, Tailwind CSS, Recharts, TanStack Query, FastAPI, pandas, scikit-learn, SQLite, ReportLab
